# Giviu — B2B Premium Corporate Gifting Platform

Live: https://giviu.vercel.app

## About
E-commerce catalog platform for premium B2B corporate gifts. 
Built as a solo project using AI-assisted development.

## Stack
- Next.js 16, React 19
- Supabase (PostgreSQL, API, RLS)
- Vercel (deployment)
- CSS Modules

## Features
- Product catalog with full specs, color variants, 
  personalization methods, MOQ
- Brand pages with SEO (JSON-LD schemas)
- Filter panel with instant filtering
- Mobile-responsive (2-column grid, swipe gestures)
- Accessibility (ARIA, keyboard navigation)

## Built with AI
This project was built using Claude as a technical 
assistant. As a non-programmer with 15 years of B2B 
product management experience, I used AI to bridge 
the gap between business knowledge and technical 
implementation.
